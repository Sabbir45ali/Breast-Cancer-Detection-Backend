# Breast Cancer Detection Backend

A professional Django REST API backend for Breast Cancer Detection using Machine Learning and Deep Learning.

This system supports:

- User authentication
- Organization authentication
- Profile management
- Image classification
- Data prediction
- Prediction history
- Cloud image storage
- Firebase authentication
- AWS deployment ready


----------------------------------

## Tech Stack

Backend:
- Django
- Django REST Framework

Database:
- Firebase Firestore

Authentication:
- Firebase Authentication (UUID Token)

Machine Learning:
- Scikit-learn (.pkl model)
- TensorFlow CNN (.keras model)

Cloud Storage:
- Cloudinary

Deployment:
- AWS EC2
